# FastAPI Clean Architecture Template (DDD + Example)

A production‑ready **FastAPI** template implementing **Clean Architecture (CA)** and **Domain‑Driven Design (DDD)**.  
Includes a simple **Task Management** example to demonstrate domain modeling, use cases, and infrastructure boundaries.

Designed to help you build scalable, maintainable, and testable backend services with clear separation of concerns.

---

## Features

- Clean Architecture (independent layers)
- Domain-Driven Design (entities, value objects, domain services)
- FastAPI for HTTP interfaces
- Repository interfaces & concrete infra implementations
- Use cases (application layer)
- Task management example domain
- Dependency injection pattern
- Config loading (environment‑based)
- Unit‑test friendly structure
- Type hints everywhere
- Ready to expand for real projects
